## 🆘 IF SOMETHING GOES WRONG

❌ Build fails with Python error:
→ Check logs: https://vercel.com/dashboard > hatobot > Logs
→ Ensure api/requirements.txt is correct
→ Check Python syntax in api/index.py

❌ Webhook returns 403:
→ Verify VERIFY_TOKEN in Vercel = hatobot-verify-token-2024
→ Check that query params match exactly

❌ Admin page shows 404:
→ Vercel > Deployments > Check build output
→ Ensure admin/package.json has build script

❌ WhatsApp not responding:
→ Missing env vars? Check Vercel project Settings > Environment Variables
→ Rerun: vercel --prod to trigger rebuild
